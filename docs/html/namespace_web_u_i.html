◆ JsHCallback()

void WebUI::JsHCallback ( Webview *  w,
const char *  args 
)

Metodo che viene chiamato a seguido della chiamata alla funzione JavaScript invoke.

È incaricato di cercare il puntatore a funzione mappato sull'oggetto map di tipo (Methods) e registrata tramite il metodo (Register) poi invocare quest'ultima ( funzione )

Questo parametro viene registrato nel metodo Create come parametro della struttura (Webview::external_invoke_cb)

Non è necessario definirla a livello di interfaccia dato che non deve essere visibile agli altri file o parti che compongono l'applicazione

Parametri
[in]wPuntatore all'oggetto istanza Webview
[in]argsArray di caratteri contenenti il buffer json serializzato contenente nome della funzione da invocare e parametri da passare a quest'ultima
Restituisce
void

Argomenti passati alla funzione deserializzati sotto forma di oggetto json
Iteratore dell'oggetto map specializzato dal tipo di dato Methods che rappresenta l'oggetto contenente il puntatore a funzione se trovato oppure che assume il valore di fine della lista
