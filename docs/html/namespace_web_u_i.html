◆ JsHCallback()

void WebUI::JsHCallback ( Webview *  w,
const char *  args 
)
inline

Metodo che viene ivocato a seguido della chiamata alla funzione Javascript window.external.invoke() da parte della webview.

La struttura JSON di riferimento è la seguente:

{
"fn": "<Namespace>::<Method>",
"params": {
...
}
}

Invoca la funzione identificata dal parametro 'fn' presente nell'oggetto JSON deserializzato tramite il puntatore a funzione mappato nella lista dei metodi di callback.

Parametri
[in]wPuntatore all'oggetto istanza Webview
[in]argsArray di caratteri contenenti il buffer json serializzato

Dati callback in formato JSON deserializzato
